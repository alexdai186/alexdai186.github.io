What's my tip-of-the-week?

My pick of the week is to use numpy whenever you need to do numerical analysis in Python. It's a great library for manipulating large amounts of data in that it provides very efficient library methods, as well as an array of useful functions that will save hours off your life. All you have to do is pip install numpy and import it into your programs and you'll have a super powerful library at your fingertips!
